# 股票大师每日复盘与热门标的分析技能 (daily_run_skill)

本技能定义了如何作为**股票大师**，结合 **agent-reach** 的网页抓取能力定位全球宏观与微观因子，并调用 **daily_stock_analysis (DSA)** 执行自动化 K线拉取、多市场共振与技术面融合的 AI 决策分析。

## 🚀 极致量化执行算法 (10次收集 + 5次调仓)

为了应对瞬息万变的全球多市场波动，本技能执行**“高频扫描、审慎决策、随机潜伏”**的极致量化算法：

```
+-----------------------------------------------------------------------------------+
|                                每日交易时间线 (9:30 - 15:00)                         |
+-----------------------------------------------------------------------------------+
|                                                                                   |
|  [早盘分析]  早上 8:00 准时抓取全球隔夜数据，生成早盘分析并推送飞书群                    |
|                                                                                   |
|  [数据收集]  S1 ---- S2 ---- S3 ---- S4 ---- S5 ---- S6 ---- S7 ---- S8 ---- S9 ---- S10
|               \      /      /      /      /                                       |
|  [综合评估]    \    /      /      /      /  (审视前5次收集结论)                     |
|                 v  v      v      v      v                                         |
|  [量化调仓]    T1 ──────> T2 ──────> T3 ──────> T4 ──────> T5                      |
|                ^          ^          ^          ^          ^                      |
|  [时间决定]  (由当前数据分析在 0 - 120 分钟内动态设定基础间隔 + 随机数扰动)          |
+-----------------------------------------------------------------------------------+
```

### 1. 每日早上 8:00 早盘分析 (Pre-Market Analysis)
每个交易日早上 8:00，系统自动触发 `daily_run_skill` 进行早盘分析：
*   **数据源：** 抓取美股隔夜收盘、中概股金龙指数（HXC）表现、新加坡 A50 期指、离岸人民币波动、隔夜原油/黄金大宗商品波动、以及最新的国内外时事政策与舆情。
*   **输出内容：** 评估今日大盘 MSS 初始分值，制定今日**“下一步预计操作”**与**“预期收益率目标”**。
*   **主动推送：** 8:05 前将精美的早盘分析 Markdown 卡片自动推送到绑定的飞书群聊。

### 2. 每日 10 次数据收集 (S1 - S10)
系统在交易日内（9:15 - 15:00）均匀或按盘口波动密集度执行 **10 次全球市场与舆情数据收集**：
*   **宏观与政策：** 抓取最新政策变动、大宗商品（原油、黄金）及美元指数/离岸人民币波动。
*   **多市场共振：** 监控美股期指、新加坡 A50、恒生指数及中概金龙指数（HXC）的日内波动。
*   **资金流向：** 监控沪深港通南北向资金的实时净流入/流出速度。
*   **社交舆情：** 抓取 V2EX、小红书、微博等平台的科技与股票舆情。

### 3. 每次量化调仓前审视前 5 次收集结论
每日最多进行 **5 次调仓量化机会 (T1 - T5)**。在执行任何买卖操作前，系统必须：
1.  读取并解析**前 5 次数据收集（S_n 到 S_n-4）**的结构化结论。
2.  计算**宏观情绪综合分值 (Macro Sentiment Score, MSS)**：
    $$MSS = w_1 \cdot \text{汇率因子} + w_2 \cdot \text{外资流向} + w_3 \cdot \text{全球共振} + w_4 \cdot \text{舆情热度}$$
3.  只有当 MSS 发生趋势性转向，或个股技术指标（MA5/MA20、20日位置）触发硬性买卖阈值时，才执行调仓。

### 4. 调仓时间由数据评估与随机数共同决定 (动态 0 - 120 分钟)
为了防止被主力资金算法“反向猎杀”（Anti-Frontrunning），每次调仓的精确时间（分/秒）不固定。
系统根据当前收集到的全球多市场数据分析，在 0 - 120 分钟内动态设定基础间隔：
$$\text{Base\_Interval} = g(\text{Market\_Volatility}, \text{Volume\_Speed}) \in [0, 120]\text{ 分钟}$$
$$\text{Next\_Trade\_Time} = \text{Base\_Interval} + f(\text{Last\_MSS}) + \text{Random\_Delay}(\text{Seed})$$
*   **Base_Interval (动态基础间隔)：** 
    *   **极速调仓模式 (0 - 30 分钟)：** 当美股/港股出现断崖式下跌、南北向资金单向流速超过 50 亿/小时、离岸人民币瞬间贬值超过 150 点时，基础间隔自动压缩至 **0 - 30 分钟**。
    *   **常规波动模式 (30 - 90 分钟)：** 当多市场共振温和，资金流速处于常规区间时，基础间隔设定在 **30 - 90 分钟**。
    *   **横盘潜伏模式 (90 - 120 分钟)：** 当全球市场横盘震荡、成交量极度萎缩、无重大宏观政策出台时，基础间隔拉长至 **90 - 120 分钟**，以静制动。
*   **Last_MSS (上次评估分值)：** 结合上次评估分值进一步微调间隔。
*   **Random_Delay (随机潜伏延迟)：** 
    *   在**常规波动模式**与**横盘潜伏模式**下，引入 **1 - 15 分钟**的随机延迟，彻底打乱交易时间分布，实现完美潜伏。
    *   在**极速调仓模式**下，为了确保在极端行情中实现毫秒级避险或抢筹，**随机潜伏延迟强制设定为 0 分钟**（即：一旦触发极速调仓，系统立刻无延迟执行买卖操作，绝不拖延一秒）。

### 5. 自动化飞书主动推送
每次交易执行完毕后，量化引擎将自动调用飞书开放平台 API，通过 `tenant_access_token` 向绑定的飞书群聊（如《每天股票量化交易》，群 ID: `oc_856923efb46d3c48405ae54ede7b784b`）推送精美的富文本 Markdown 卡片简报，实现无感、主动监控。

## 🛠️ 运维与排障指南

### 1. 提示 "LLM API Key 未配置"
*   **原因：** Cursor Cloud Agent 运行在隔离沙箱中，新配置的 Secrets 无法在当前热会话中生效。
*   **解决办法：** 
    1. 确保在 [Cursor Dashboard → Cloud Agents → Secrets](https://cursor.com/dashboard/cloud-agents) 页面中配置了 `GEMINI_API_KEY` 或 `OPENAI_API_KEY`。
    2. **重启当前 Agent 任务**，使 Secrets 环境变量成功注入。

### 2. Efinance 历史 K 线接口失败 (RemoteDisconnected)
*   **原因：** 东方财富接口对高频连续请求有随机熔断限制。
*   **解决办法：** DSA 内部已集成多数据源自动切换。当 `EfinanceFetcher` 熔断后，系统会自动切换到 `TencentFetcher`（腾讯接口）进行兜底拉取，无需人工干预。
